AI chatbot with knowledge graph, theme sync, and OpenAI-compatible providers.

== Description ==

AI Assistant with Knowledge Graph is a full-featured enterprise WordPress plugin providing:

* Intelligent theme synchronization with your WordPress site
* OpenAI-compatible AI provider with dynamic model fetching
* Website content indexing and knowledge brain
* Obsidian-style knowledge graph visualization
* Smart fallback and human escalation
* English/Arabic multi-language with RTL support
* Analytics dashboard and compliance audit trail
* Gutenberg block, shortcode, and Elementor widget

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. Go to AI Assistant > AI Provider and configure your API key
4. Click "Fetch Models" to load available models
5. Sync website content from the Dashboard

== Shortcode ==

`[ceac_chatbot]` - Embed chatbot inline

== Environment Variables ==

`CEAC_API_KEY` - Override API key
